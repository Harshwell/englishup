# data_sources

Direktori ini dipakai untuk menyimpan dataset mentah yang menjadi fondasi pipeline materi v3.

Target sumber data:
- CEFR-J / Open Language Profiles
- Words-CEFR Dataset
- SUBTLEX-US
- Tatoeba exports
- ELLIPSE Corpus

Catatan penting:
- Jangan commit dataset besar mentah ke repo publik tanpa review lisensi.
- Simpan metadata, mapping, atau sampel kecil bila memang diperlukan untuk demonstrasi.
- Gunakan workflow GitHub Actions atau proses lokal untuk menormalisasi dataset menjadi JSON yang lebih ringan.
